# Titan M Research Tool

Security research tool for Google Pixel Titan M secure element.

## Features

- Analyze Titan M communications
- AES-GCM encryption/decryption
- Fastboot verification check
- Part of the NullKia mobile research project

## Installation

```bash
cargo install titan_m
```

## Usage

```bash
titan_m <command> [options]
```
